# Velvet Brew Platform

A premium cafe ordering system and POS built for Velvet Brew.

## Features
- Dynamic Orders tracking (Customer)
- Hold & Send to Kitchen functionality
- Kitchen Display System (KDS)
- Point of Sale (POS) System for Admins
- Secure Auth and Supabase PostgreSQL Database

## Setup
1. Clone repository
2. Change into `frontend` and run `npm install`
3. Change into `backend` and run `npm install`
4. Copy env configs
5. Run `npm run dev` in both folders
